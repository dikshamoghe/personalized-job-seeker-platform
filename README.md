# AI-Powered Personalized Job Search and Application Tracking Platform

full-stack web application designed to help job seekers discover job opportunities, manage applications, and receive personalized career recommendations. The platform provides a centralized dashboard where users can track their job application progress, manage resumes, and find opportunities based on their skills and preferences.

---

## Project Objectives

- Provide the latest recruitment updates for freshers and experienced professionals.
- Help users manage and track job applications in one place.
- Recommend relevant jobs based on user profiles and resumes.
- Simplify the job search process using smart filters.
- Build a scalable platform with modern web technologies.

---

## Core Features (MVP)

- User Authentication (JWT)
